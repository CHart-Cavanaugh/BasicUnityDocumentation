Introduction

What Is Unity?

Unity is a popular game engine (a development environment used by developers to make video games for different platforms). Besides being useful for making video games, Unity can also be used to make regular applications (such as applications on desktop and mobile devices).

Some examples of video games that were made using Unity include:

  • Cuphead (2017)
  • Among Us (2018)
  • Subnautica (2018)
  • Fall Guys: Ultimate Knockout (2020)
  • Ori and the Will of the Wisps (2020)

Installing Unity And Visual Studio Code

Section Introduction

In order to work on projects in Unity, developers need to have: the necessary programs downloaded and installed, an account with Unity, and a license added to their Unity account. Unity offers different licenses in order for developers to use their editor (which includes a free license to get started with). To obtain a license for using Unity, developers need to sign up for a Unity account.

The necessary programs that developers need in order to work on projects in Unity are:

  • Unity Hub
  • Some version of the Unity editor
  • An IDE that can be used with the Unity editor (preferably Visual Studio Code)

Read through the section for more information on: downloading and installing Unity Hub, installing Unity through Unity Hub, creating an account with Unity and adding a license to that account, and downloading and installing Visual Studio Code.
