Introduction

What Is Unity?

Unity is a popular game engine (which is a development environment used by developers to make video games for different platforms). Besides being useful for making video games, Unity can also be used to make regular applications (such as applications on desktop and mobile devices).

Some examples of video games that were made using Unity include:

  • Cuphead (2017)
  • Among Us (2018)
  • Subnautica (2018)
  • Fall Guys: Ultimate Knockout (2020)
  • Ori and the Will of the Wisps (2020)
